## Overview
An interactive portfolio tracking dashboard built with Python and Streamlit.
Users can dynamically build portfolios, fetch live market data, and analyze
returns and risk metrics.

## Features
- Interactive stock selection
- Live price fetching via yfinance
- Portfolio valuation and returns
- Risk metrics (Volatility, Sharpe Ratio)
- Asset allocation visualization

## Tech Stack
- Python
- Streamlit
- Pandas / NumPy
- yfinance
- Plotly
- SQLite (optional persistence)
